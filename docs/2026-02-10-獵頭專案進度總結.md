**檔案位置**: `~/clawd/hr-tools/jd-manager.sh`

```bash

./jd-manager.sh list

# 搜尋職缺
./jd-manager.sh search "AI"

./jd-manager.sh update 2 "已成交"

# 統計報表
./jd-manager.sh report
```
