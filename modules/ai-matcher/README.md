# AI 配對系統 (AI Matcher)

## 功能

智能職缺配對系統，計算候選人與職缺的匹配度，評級為 P0/P1/P2。

## 核心檔案

- `ai_matcher_v2.py` - 配對引擎（v2 版本）

## 使用方式

```bash
python ai_matcher_v2.py \
  --jd "職缺描述" \
  --candidate "候選人資訊" \
  --output match-result.json
```

## 評級標準

| 等級 | 分數 | 說明 |
|------|------|------|
| P0 | 80-100 | 強力推薦（高度匹配） |
| P1 | 60-79 | 推薦（中度匹配） |
| P2 | 40-59 | 可考慮（基本匹配） |
| - | <40 | 不推薦 |

## 配對因素

1. **技能匹配** (40%)
2. **年資匹配** (20%)
3. **學歷匹配** (15%)
4. **工作穩定性** (15%)
5. **產業經驗** (10%)
