# Persona Matching System - 人才畫像 + 公司畫像匹配系統

**版本**：v1.0  
**最後更新**：2026-02-23

---

## 📋 系統概述

本模組實現**人才畫像（Candidate Persona）** 與 **公司畫像（Company Persona）** 的自動生成與匹配分析。

### 核心概念

```
人才畫像 = 這個人是誰
公司畫像 = 這個地方是什麼
匹配 = 他適不適合在這裡成長

適配度 = 技能匹配 + 成長匹配 + 文化匹配 + 動機匹配
```

---

## 🎯 三大核心模組

### 1. 人才畫像生成器（Candidate Persona Generator）
- **輸入**：候選人履歷（JSON）
- **輸出**：結構化人才畫像
- **腳本**：`generate-candidate-persona.py`

### 2. 公司畫像生成器（Company Persona Generator）
- **輸入**：職缺描述 + 公司資訊（JSON）
- **輸出**：結構化公司畫像
- **腳本**：`generate-company-persona.py`

### 3. 畫像匹配分析器（Persona Matcher）
- **輸入**：人才畫像 + 公司畫像
- **輸出**：適配度評分 + 風險分析 + 建議
- **腳本**：`match-personas.py`

## 🎯 匹配模型

### 適配度計算公式

```python
適配度 = (
  技能匹配 * 0.35 +
  成長匹配 * 0.25 +
  文化匹配 * 0.25 +
  動機匹配 * 0.15
)
```

### 1. 技能匹配（35%）

```python
技能匹配 = (
  技能組合匹配 * 0.50 +
  能力層級匹配 * 0.30 +
  產業背景匹配 * 0.20
)
```

**評分標準**：
- 100分：技能完全符合，能力層級達標，產業背景完全契合
- 80分：核心技能符合，能力層級達標，產業背景部分契合
- 60分：部分技能符合，能力層級接近，產業背景有關聯
- <60分：技能不足或產業背景不符

### 2. 成長匹配（25%）

```python
成長匹配 = (
  職涯路徑匹配 * 0.50 +
  學習機會匹配 * 0.30 +
  晉升速度匹配 * 0.20
)
```

**評分標準**：
- 100分：職涯路徑完全符合候選人期望，成長空間大
- 80分：職涯路徑大致符合，有明確成長機會
- 60分：職涯路徑可接受，成長機會有限
- <60分：職涯路徑不符或無成長空間

### 3. 文化匹配（25%）

```python
文化匹配 = (
  工作風格匹配 * 0.40 +
  公司階段匹配 * 0.30 +
  用人風格匹配 * 0.30
)
```

**評分標準**：
- 100分：工作風格完全契合，公司階段符合偏好
- 80分：工作風格大致契合，可適應公司文化
- 60分：工作風格有衝突但可調整
- <60分：工作風格嚴重不符

### 4. 動機匹配（15%）

```python
動機匹配 = (
  主要動機滿足度 * 0.60 +
  不適配條件檢查 * 0.40
)
```

**評分標準**：
- 100分：主要動機完全滿足，無不適配條件觸發
- 80分：主要動機大致滿足，不適配條件可接受
- 60分：部分動機滿足，有輕微不適配
- <60分：動機不符或觸發不適配條件

---

## 🚦 匹配結果分級

```
S 級（90-100分）: 完美匹配 - 強烈推薦
A 級（80-89分）: 高度匹配 - 推薦
B 級（70-79分）: 中度匹配 - 可考慮
C 級（60-69分）: 低度匹配 - 需評估
D 級（<60分）: 不匹配 - 不推薦
```

## 🛠️ 使用方式

### 1. 生成候選人畫像

```bash
python3 generate-candidate-persona.py \
  --resume candidate-resume.json \
  --output candidate-persona.json
```

### 2. 生成公司畫像

```bash
python3 generate-company-persona.py \
  --job job-description.json \
  --company company-info.json \
  --output company-persona.json
```

### 3. 執行匹配分析

```bash
python3 match-personas.py \
  --candidate candidate-persona.json \
  --company company-persona.json \
  --output match-report.json
```

### 4. 批量匹配（一個職缺 vs 多個候選人）

```bash
python3 batch-match.py \
  --company company-persona.json \
  --candidates candidates-folder/ \
  --output batch-match-report.json
```

## 🔄 與現有系統整合

### 整合點 1：Talent Grading（人才評級）

```
人才畫像 → 輸入
  ↓
Talent Grading（S/A+/A/B/C）→ 候選人整體素質
  ↓
Persona Matching → 與特定公司/職缺的適配度
```

**差異**：
- **Talent Grading**：候選人本身的質量（與職缺無關）
- **Persona Matching**：候選人與特定公司/職缺的契合度

### 整合點 2：AI Matcher v2

```
舊系統：AI Matcher v2（技能 + 年資 + 學歷 + 穩定度）
  ↓
新系統：Persona Matching（技能 + 成長 + 文化 + 動機）
  ↓
未來：兩者結合（硬技能 + 軟實力 + 文化適配）
```
