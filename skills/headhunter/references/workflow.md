# 獵頭工作流程執行指南

## 角色分工

| 任務類型 | 執行者 | 工具 |
|---------|--------|------|
| 分析、報告、模板 | YQ1 | Prompt 模板 |
| 發信、提醒、追蹤 | YQ2 | gog (Email) |
| 電話、拜訪、談判 | Jacky | 人工 |

---

## 工作流程

### 階段 2️⃣ 人才搜尋
```
Jacky：「這是新的 JD，幫我分析」

YQ1 執行：
1. 使用 [人才畫像] prompt
2. 使用 [人才搜尋策略] prompt
3. 回覆：畫像 + Boolean String + 目標公司
```

### 階段 3️⃣ 初步篩選
```
Jacky：「這份履歷跟 JD 匹配嗎？」

YQ1 執行：
1. 使用 [履歷-JD 匹配] prompt
2. 回覆：匹配度 + 優缺點 + 建議問題
```

### 階段 4️⃣ 人選經營
```
Jacky：「幫我整理這個候選人的摘要」

YQ1 執行：
1. 使用 [候選人摘要] prompt
2. 回覆：姓名/現職/亮點/動機/期望
```

### 階段 5️⃣ 推薦客戶 ⭐
```
Jacky：「推薦這個人給 hr@client.com」

YQ1 執行：
1. 使用 [候選人摘要] prompt（如還沒做）
2. 使用 [推薦信] prompt
3. 交給 YQ2

YQ2 執行：
1. 用 gog 發 Email（附履歷）
2. 設定 3 天後跟進提醒
```

### 階段 6️⃣ 面試協調
```
Jacky：「客戶要面試這個人，時間是 2/15 下午 2 點」

YQ1 執行：
1. 使用 [面試問題] prompt
2. 準備面試資料給候選人

YQ2 執行：
1. 發面試邀請 Email
2. 設定面試前一天提醒
```

### 階段 8️⃣ 報到跟進
```
Jacky：「XXX 今天報到，設定自動跟進」

YQ2 執行：
1. 設定 Day-1 關懷信（立即）
2. 設定 Week-1 Check-in（7 天後）
3. 設定 Month-1 Check-in（30 天後）
4. 設定保證期提醒（90 天後）
```
