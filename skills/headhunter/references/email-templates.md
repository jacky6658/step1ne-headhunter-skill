# Email 模板庫

## 1. 候選人推薦信（給客戶）

```
主旨：【Step1ne】推薦 {職位} 人選 - {候選人姓名}

{客戶稱呼} 您好，

針對貴司 {職位名稱} 職缺，推薦以下人選供參考：

📋 **候選人摘要**
• 現職：{公司} / {職稱}
• 年資：{N} 年相關經驗
• 學歷：{學校} / {科系}

💡 **推薦理由**
1. {核心優勢 1}
2. {核心優勢 2}
3. {核心優勢 3}

📎 詳細履歷請參閱附件。

如有興趣安排面試，請告知方便的時間，我會協助安排。

Best regards,
Jacky Chen
Step1ne Consulting
```

## 2. 開發信（給候選人）

### 版本 A：簡短直接
```
主旨：{公司} {職位} 機會 - 想與您聊聊

{候選人姓名} 您好，

我是 Step1ne 的 Jacky，看到您在 {現職公司} 擔任 {職稱} 的經歷，想與您分享一個 {職位} 的機會。

這是一間 {公司簡介}，目前在找 {關鍵需求}。

方便這週聊 15 分鐘嗎？

Best,
Jacky
```

### 版本 B：詳細價值導向
```
主旨：{職位} 機會 @ {公司} - 您的 {技能} 經驗很符合

{候選人姓名} 您好，

我是 Step1ne 的 Jacky，專注於 {產業} 人才招募。

看到您在 {現職公司} 的經歷，特別是 {具體經驗/專案}，覺得很符合目前在找的 {職位}。

【關於這個機會】
• 公司：{公司名稱}（{簡介}）
• 職位：{職位名稱}
• 亮點：{1-2 個吸引人的點}

【為什麼找您】
您在 {技能/領域} 的經驗，正是這個職位需要的。

有興趣了解更多嗎？我們可以先聊 20 分鐘。

Best regards,
Jacky Chen
```

## 3. 面試邀請（給候選人）

```
主旨：{公司} {職位} 面試邀請 - {日期}

{候選人姓名} 您好，

恭喜！{公司} 對您的履歷非常有興趣，想邀請您進行面試：

📅 **面試資訊**
• 日期：{日期}
• 時間：{時間}
• 地點：{地點 / 線上連結}
• 面試官：{面試官姓名/職稱}
• 預計時間：{N} 分鐘

📝 **建議準備**
1. 了解公司背景（官網：{網址}）
2. 準備 1-2 個想問的問題
3. 回顧您在 {相關經驗} 的具體案例

如有任何問題，隨時聯繫我。

祝面試順利！

Best,
Jacky
```

## 4. 面試跟進（問候選人反饋）

```
主旨：{公司} 面試後續 - 想聽聽您的想法

{候選人姓名} 您好，

今天的面試辛苦了！想了解一下您的感受：

1. 整體面試體驗如何？
2. 對這個職位還有興趣嗎？
3. 有沒有想補充或想問的？

我會持續跟進客戶那邊的反饋，有消息第一時間告訴您。

Best,
Jacky
```

Jacky
