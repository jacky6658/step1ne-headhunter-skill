## 通知時機

### 自動通知（每次執行後）
- ✅ 每天 09:00, 14:00, 20:00 執行後自動發送
- ✅ 包含執行結果統計
- ✅ Top 3 推薦候選人

### 手動通知（測試用）
```bash
openclaw message send \
  --target "-1003231629634" \
  --thread-id 304 \
  --message "測試訊息"
```

### 錯誤通知（執行失敗時）
- ❌ 自動發送錯誤通知
- ❌ 包含錯誤訊息和日誌檔案位置
- ❌ 建議檢查步驟

## 通知格式規範

### Emoji 使用
- ✅ 成功執行
- ⚠️ 警告（找到候選人數量少）
- ❌ 執行失敗
- 🔍 搜尋中
- 📊 統計資料
- 💼 職缺資訊
- 👤 候選人資訊

### 連結格式
- GitHub: `https://github.com/username`
- LinkedIn: `https://linkedin.com/in/username`
- Google Sheets: 完整 URL（太長可縮短）

### 時間格式
- 執行時間：HH:MM（例：09:00）
- 日期：YYYY-MM-DD（例：2026-02-23）

### 數字格式
- 候選人數量：XX 位
- 評分：XX 分
- 年資：X 年

## 測試通知

手動發送測試通知：
```bash
openclaw message send \
  --target "-1003231629634" \
  --thread-id 304 \
  --message "$(cat <<EOF
✅ Sourcing Bot 測試通知

這是一則測試訊息，確認：
• Emoji 正確顯示
• 換行正常
• 連結可點擊：https://github.com/test

狀態：✅ 測試成功
EOF
)"
```
